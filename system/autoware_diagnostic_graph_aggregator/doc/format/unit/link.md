# Link

The `link` object is a unit that refers to another unit.

## Format

| Name   | Type     | Required | Description                            |
| ------ | -------- | -------- | -------------------------------------- |
| `type` | `string` | yes      | Specify `link` when using this object. |
| `link` | `string` | yes      | The path of the unit to reference.     |
