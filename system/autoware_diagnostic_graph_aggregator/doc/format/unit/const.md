# Constant

The constant object is a unit with a fixed error level.

## Format

| Name   | Type     | Required | Description                                 |
| ------ | -------- | -------- | ------------------------------------------- |
| `type` | `string` | yes      | Specify error level when using this object. |

## Error levels

The supported error levels are as follows.

- `ok`
- `warn`
- `error`
- `stale`
