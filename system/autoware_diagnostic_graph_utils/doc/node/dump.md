# Dump tool

This tool displays `/diagnostics_graph` in table format.

## Usage

```bash
ros2 run autoware_diagnostic_graph_utils dump_node
```

## Examples

Terminal 1:

```bash
ros2 launch diagnostic_graph_aggregator example-main.launch.xml
```

Terminal 2:

```bash
ros2 run autoware_diagnostic_graph_utils dump_node
```
